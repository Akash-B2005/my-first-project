# 🎬 Netflix Clone (Static Website)

A simple and responsive Netflix landing page clone built using **HTML, CSS, and JavaScript**. This project replicates the basic UI of Netflix for learning and practice purposes.

---

## 🚀 Features

* 🔥 Modern Netflix-style UI
* 📱 Fully responsive design
* 🎥 Hero banner section
* 🎞️ Movie thumbnails section
* 📩 Email input with validation
* ⚡ Smooth hover effects and animations

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
Netflix-Clone/
│── index.html
│── style.css
│── script.js
```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/netflix-clone.git
```

2. Open the project folder

3. Run the project:

* Double click `index.html`
  OR
* Open with Live Server (VS Code recommended)
